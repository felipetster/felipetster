# Olá, sou o Felipe!

Minha jornada na tecnologia começou na análise de dados para auditoria na **EY (Ernst & Young)**. Lá, meu trabalho era investigar grandes volumes de informação, identificar inconsistências e garantir a conformidade com regras de negócio críticas.

Com o tempo, percebi que essa busca por precisão e prevenção de falhas era minha verdadeira motivação. A transição para **Quality Assurance (QA)** foi natural. Hoje, uno meu background analítico com habilidades de codificação para garantir a qualidade de software, não apenas encontrando bugs, mas validando a lógica do negócio e a integridade dos dados.

---

## No que estou focado agora

### Automação de APIs (Backend First)
- **Stack:** Postman + JavaScript (ES6+).
- **Diferencial:** Desenvolvimento de scripts avançados utilizando métodos como `.reduce()` e `.find()` para auditar cálculos financeiros no server-side.
- **Cenários:** Automação de fluxos completos (`Auth JWT` → `CRUD` → `Validação de Banco de Dados`) com manipulação de variáveis dinâmicas de ambiente.

### Automação Web (E2E)
- **Stack:** Python, Selenium e Pytest.
- **Escopo:** Automatização de fluxos críticos como Login, Carrinho de Compras e Checkout, focando na estabilidade dos testes e Page Object Model.

### Testes de Performance & Carga
- **Ferramenta:** JMeter.
- **O que analiso:**
  - **Load Testing:** Comportamento da aplicação sob carga esperada.
  - **Stress Testing:** Identificação do ponto de ruptura (Breakpoint) do sistema.
  - **Métricas Chave:** Análise crítica de *Throughput* (Vazão), *Latency* (Latência de Rede) e *Response Time* (P90/P95) para identificar gargalos antes que afetem o usuário final.

### Gestão da Qualidade
- Documentação técnica de bugs e evidências rastreáveis utilizando **Jira/Zephyr**.
- Priorização baseada em risco e impacto no negócio.
