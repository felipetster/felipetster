# Olá, sou o Felipe!

Minha jornada na tecnologia começou na análise de dados para auditoria na **EY**. Lá, meu trabalho era como basicamente de um detetive: investigar grandes volumes de informação, encontrar a inconsistência, o erro e garantir que tudo estivesse em conformidade com as regras de negócio.
