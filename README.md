# Olá, sou o Felipe!

### QA Analyst | Test Automation | Data Background

Minha jornada na tecnologia começou na análise de dados para auditoria na **EY (Ernst & Young)**. Nesse contexto, atuei na validação de grandes volumes de dados, identificação de inconsistências e garantia de conformidade com regras de negócio críticas.

Com o tempo, percebi que essa busca por precisão, prevenção de falhas e entendimento do impacto no negócio se conecta diretamente com **Qualidade de Software (QA)**. Hoje, aplico meu background analítico aliado à automação de testes para garantir não apenas o funcionamento do software, mas a consistência das regras e a integridade dos dados.

---

## No que estou focado agora

### Automação de APIs (Backend First)
* **Stack:** Postman + JavaScript (ES6+).
* **Atuação:** Criação de scripts com lógica condicional para validação de regras de negócio e cálculos financeiros no backend.
* **Cenários:** Automação de fluxos completos (Autenticação JWT, operações CRUD e validação de dados retornados), utilizando variáveis dinâmicas de ambiente.

### Automação Web (E2E)
* **Stack:** Java, Selenium WebDriver e TestNG.
* **Escopo:** Automação de fluxos críticos ponta a ponta, como Login e Solicitação de Crédito (ex: Projeto CrediFlow), aplicando **Page Object Model** e boas práticas para garantir estabilidade e manutenção.

### Testes de Performance e Carga
* **Ferramenta:** JMeter.
* **Foco:** Load Testing e Stress Testing.
* **Análise:** Avaliação de métricas como Throughput, Latência e Response Time (P90/P95) para identificação de gargalos.

### Gestão da Qualidade
* Criação e documentação de casos de teste, bugs e evidências rastreáveis (Jira e Zephyr).
* Priorização de defeitos baseada em risco e impacto no negócio.
