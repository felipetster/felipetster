## O Que Eu Faço

**Testes Web End-to-End (E2E)** Automação de jornadas críticas do usuário utilizando **Cypress (JavaScript)** e **Selenium WebDriver (Java/Python)**, aplicando padrões de arquitetura como Page Object Model para garantir testes escaláveis e de fácil manutenção.

**Automação com Foco no Backend & Dados** Testes de API com Postman + JavaScript/Python. Validação avançada de integridade de banco de dados com **SQL**, além da construção de scripts para regras de negócio, cálculos financeiros e fluxos de autenticação (JWT).

**Engenharia de Qualidade com IA** Utilização de Inteligência Artificial (LLMs) como ferramenta estratégica para acelerar a análise de causa raiz, identificar padrões de falhas e criar relatórios automatizados. O foco é utilizar a IA para otimizar a produtividade, reduzir o tempo de triagem de bugs e devolver horas úteis ao time de desenvolvimento, elevando a qualidade final do software.

**Integração Contínua & DevOps (CI/CD)** Configuração de pipelines de **CI/CD** para garantir a execução automatizada de testes a cada deploy, utilizando **Docker** para criar e orquestrar ambientes de teste isolados (microsserviços e bancos de dados).

**Testes de Performance & Gestão** Testes de carga com JMeter (análise de P90/P95). Criação de casos de teste, rastreamento de defeitos (Jira + Zephyr) e priorização baseada em riscos corporativos.
