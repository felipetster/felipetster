## O Que Eu Faço

**Testes Web End-to-End (E2E)** Automação de jornadas críticas do usuário utilizando **Cypress (JavaScript)** e **Selenium WebDriver (Java)**, aplicando padrões de arquitetura como Page Object Model para garantir testes escaláveis e de fácil manutenção.

**Automação com Foco no Backend** Testes de API com Postman + JavaScript. Construção de scripts de validação condicional para regras de negócio, cálculos financeiros e fluxos completos de autenticação (JWT, operações CRUD).

**Integração Contínua & DevOps (CI/CD)** Configuração de pipelines de **CI/CD** para garantir a execução automatizada de testes a cada deploy, utilizando **Docker** para criar e orquestrar ambientes de teste isolados (microsserviços e bancos de dados).

**Testes de Performance** Testes de carga e estresse com JMeter. Análise de métricas de vazão (throughput), latência e tempo de resposta (P90/P95) para identificar gargalos.

**Gestão de QA** Criação de casos de teste, rastreamento de defeitos (Jira + Zephyr) e priorização baseada em riscos.
