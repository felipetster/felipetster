## O Que Eu Faço

**Testes Web End-to-End (E2E)**
Automação de jornadas críticas do usuário utilizando **Cypress (JavaScript)** e **Selenium WebDriver (Java)**, aplicando padrões de arquitetura como Page Object Model para garantir testes escaláveis e de fácil manutenção.

**Automação com Foco no Backend & Dados**
Testes de API com Postman, incluindo manipulação de tokens JWT e validação de fluxos de autenticação. Validação de integridade de banco de dados com **SQL**, além da construção de scripts para regras de negócio e cálculos de dados.

**Engenharia de Qualidade com IA**
Utilização de Inteligência Artificial (LLMs) como ferramenta estratégica para acelerar a geração e revisão de casos de teste, identificar padrões de falhas e apoiar a documentação. O foco é usar IA para otimizar produtividade e reduzir tempo de triagem, sempre com curadoria humana sobre o resultado.

**Integração Contínua (CI)**
Configuração de pipelines de **CI** com GitHub Actions para execução automatizada de testes a cada push/PR, utilizando **Docker** para criar ambientes de teste isolados.

**Testes de Performance & Gestão**
Noções de teste de carga com JMeter. Criação de casos de teste, rastreamento de defeitos (Jira + Zephyr) e priorização baseada em risco.
